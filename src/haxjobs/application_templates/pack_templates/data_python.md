# Python Data and Analytics Developer Pack Template

family: data_python
cv_variant: data_python
manual_review_required: true
auto_submit: false
cover_letter_template: cover_letters/data_python.md

## Pack purpose
Create job-specific prep material while reusing the stable `data_python` CV variant.

## Inputs required from job/evaluator
- company
- role_title
- source_url or apply_url
- hiring_manager_or_team, if found from a reliable source
- jd_match_points
- company_reason
- evidence_story
- gap_note
- fit_score
- fit_verdict

## Output files
- fit_report.md
- cover_letter.md
- field_answers.md
- interview_questions.md
- telegram_summary.md
- metadata.json

## field_answers
Keep answers short, truthful, and copy-paste friendly. Use the same voice: natural, confident, not stiff. Legal, visa, demographic, disability, sponsorship, and salary answers must come from locked profile facts or Arinze approval.

## Review rules
- CV is referenced, not regenerated.
- Cover letter is generated from the role template and JD facts.
- No application is submitted automatically.
- Any uncertain claim goes into review notes, not the submitted letter.
