## Key Decisions Made

1. **DB is single source of truth.** Intake JSON is write-only audit trail. No more dual reading.
2. **Unskip means re-evaluate with enriched profile.** Not create duplicate.
3. **Approve means bypass evaluation entirely.** Status = "approved", ready for pack generation.
4. **Current stack stays.** No FastAPI migration yet. `http.server` works for this scale.
5. **React Router added.** Real URLs, browser back button, deep linking.
6. **No external UI libraries.** Toasts are custom. Icons are inline SVGs. Zero npm bloat.
7. **Evaluation history preserved.** Old scores archived when re-evaluating, not lost.
