# Standalone utility scripts

These are not called by the pipeline. They're manual tools for development and deployment.

- `archilles-webui` — manage Archilles VPS web UI
- `dash-tunnel` — SSH tunnel for remote dashboard access
- `debug_job_pipeline.py` — inspect pipeline stage state for debugging
- `generate-pack` — manually generate a pack for a specific job
- `haxjobs-update` — pull latest code on Archilles VPS
- `pull-cv-variants` — pull CV variant files from source
- `render-cv-variant` — render a CV variant HTML to PDF
- `review-pack` — review a generated pack output
- `seed-cv-variants-from-packs` — seed CV variants database from existing packs
- `update-archilles-private` — update Archilles private repo
